class  ADFuncVector
 ADFuncVector represents a vector of functions and performs automatic differentiation on this vectorized functions as a whole. It is a wrapper class of std::vector<ADFunc> to make taking partial derivative from vectorized functions easier (we write the for-loops of .dval_wrt() for you so all you need to do is declaring a ADFuncVector and calling its member functions)! You have many options of taking partial derivatives of a ADFuncVector object, whether you want a scalar number, a 1-d vector, or a 2-D matrix. We also support a function of getting Jacobian matrix for your convenience. More...
